# 미니 룰렛 기능 강화 기획서

## 1. 룰렛 화면 내 리스트 전환 기능
사용자가 관리 탭으로 이동하지 않고도 현재 활성화된 룰렛 리스트를 즉시 변경할 수 있는 기능을 추가합니다.

### UI/UX
- **위치**: `RouletteView` 상단 타이틀 영역.
- **형태**: 타이틀 옆에 드롭다운 아이콘 또는 `Select` 박스 배치.
- **동라**: 
    - 클릭 시 현재 보유 중인 모든 룰렛 리스트(`RouletteList`)의 이름 목록 표시.
    - 리스트 선택 시 `selectedListId`를 업데이트하여 즉시 화면 갱신.
    - 스핀 중(`isSpinning === true`)일 때는 비활성화(Disabled)하여 오작동 방지.

## 2. 전체 데이터 내보내기 및 가져오기 (백업/복구)
애플리케이션의 모든 데이터를 JSON 파일 형태로 백업하고 복구하는 기능입니다.

### 기능 명세
- **내보내기 (Export All)**
    - **대상 데이터**: `lists` (모든 리스트), `settings` (앱 설정).
    - **포맷**: JSON 파일.
    - **파일명**: `mini-roulette-backup-YYYYMMDD.json`.
- **가져오기 (Import All)**
    - **동작**: JSON 파일을 업로드하여 `useRouletteStore`의 상태를 덮어쓰기(Overwrite) 또는 병합(Merge). (초기 구현은 덮어쓰기로 진행 후 필요 시 병합 옵션 고려).
    - **유효성 검사**: 필수 필드(`lists`, `id`, `items` 등) 확인 후 유효하지 않은 파일이면 에러 처리.

## 3. 아이템 대량 추가 기능 (Bulk Import)
여러 항목을 텍스트로 한 번에 입력받아 리스트에 추가합니다.

### UI/UX
- **위치**: `ListManager`의 아이템 추가 영역 옆에 '대량 추가(Bulk Add)' 버튼 배치.
- **인터랙션**:
    1. 버튼 클릭 시 모달 또는 확장 영역 오픈.
    2. `textarea`에 텍스트 입력 (엔터로 구분).
    3. '추가' 버튼 클릭 시 줄바꿈(`\n`) 기준으로 분할하여 현재 리스트에 일괄 추가.
    4. 빈 줄은 무시.

---

### 기대 효과
- **편의성**: 리스트 간 빠른 이동으로 사용성 증대.
- **데이터 안전**: 전체 백업을 통해 브라우저 캐시 삭제 등의 상황 대비.
- **효율성**: 대량의 데이터를 손쉽게 입력 가능.
